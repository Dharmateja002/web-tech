# Q6 - MongoDB Schema + CRUD

This folder contains Mongo shell scripts to create a collection (schema via validation) and demonstrate CRUD.

## How to run (Mongo Shell)
1. Start MongoDB.
2. Open `mongosh`.
3. Run the script:
   - `load("crud.js")`

## Files
- `crud.js`: creates `students` collection with validation + demonstrates insert/find/update/delete
